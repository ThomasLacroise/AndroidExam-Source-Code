PGR208 - Shopping APP (Android)
-
Dette prosjektet ble utviklet som en del av eksamen i PGR208 - Android Development ved Høyskolen Kristiania - høst 2023.

Oppgaven gikk ut på å utvikle en shopping-app for Android, hvor brukeren kan bla gjennom produkter hentet fra et eksternt API, lese om produktet, legge produktet til i en handlekurv, og simulere et kjøp.
<hr style="height": 3px>

Teknologier
-
  - Kotlin
  - Android Studio
  - Jetpack Compose
  - REST API
  - State Management

<hr style="height": 3px>

Funksjonalitet
- Hente produkter fra ekstern API
- Vise produkter i en scrollable liste
- Trykke på produkt og få informasjon
- Legge produkt til i handlekurv
- Vise handlekurv med oversikt
- Simulere kjøp av produkter i handlekurv
- Vise ordrehistorikk
- Navigasjon mellom forskjellige views
- Responsiv og enkel UI med Jetpack Compose

<hr style="height": 3px>

Kjøre prosjektet
-
1. Åpne prosjektet i Android Studio
2. Kjør appen på simulator eller ønsket fysisk Android-enhet

<hr style="height": 3px>

Om prosjektet
-
Prosjektet viser hvordan man kan bygge en enkel shopping-app i Android med moderne teknologi som Jetpack Compose.

Appen integrerer med et REST API for å hente produktet og gir brukeren mulighet til å se på produkter, legge de til i en handekurv og gjennomføre et "kjøp".
